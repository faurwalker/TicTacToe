# TicTacToe
Simple implementation of Tic Tac Toe game written on c.

## Instalation
Clone repository using `git clone`

```bash
git clone https://github.com/faurwalker/TicTacToe
```
To build use `make` 
```shell make ```

## Usage 

**-h** use to get help on the commands
